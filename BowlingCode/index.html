Player Score Table

This table application allows a player to enter scores per frame of a bowling game
to keep track of scores for each frame as well as final score
1. Click or press the number corresponding to the pins knocked on each throw.
2. Score will de displayed for each frame and total score will be displayed at end of 10th frame.
